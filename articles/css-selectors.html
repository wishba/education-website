CSS: Selectors

Posted by Hafizh Khalis - 05 Agustus 2023

Sebelumnya kita sudah mempelajari apa itu css, dan bagaimana mekanisme kerja css. Pada artikel ini, kita akan mempelajari selektor apa saja yang dapat digunakan untuk membuat style pada element html.

Selektor sendiri merupakan kunci atau simbol yang dapat digunakan pada css untuk menyeleksi dan memilih element HTML.

Ada 5 macam selektor dasar pada css yaitu:

  • Selektor Tag
  • Selektor Class
  • Selektor ID
  • Selektor Atribut
  • Selektor Universal

1. Selektor Tag

Selektor tag merupakan selektor yang menggunakan tag element html seperti body, p, h1,span untuk menyeleksi sebuah element yang akan diberikan css property, contoh penggunaannya:

2. Selektor Class

Class merupakan selektor yang dibuat sesuai kebutuhan, dapat dikatakan class merupakan selektor yang penamannya bisa dibuat oleh developer, jadi setiap tag dapat disesuaikan kebutuhan berdasarkan class yang diseleksi

Untuk menggunakan class selektor, perlu menambahkan atribut class pada tag yang ingin diberikan class, lalu value dapat diisi dengan nama tag. Contoh penggunannya seperti berikut:

See the Pen Class Selectors by Hakhajid (@hafizhkhalis) on CodePen.

Lalu pada cssnya dapat diberikan tanda titik (.) diawal untuk menandakan bahwa element yang kita seleksi menggunakan class.

3. Selektor ID

ID merupakan selektor yang hampir sama cara penggunaannya dengan class selektor, perbedaannya ada pada tingkat prioritas dan jumlah element yang dapat diberikan id. Berbeda dengan class yang dapat digunakan pada banyak element, id hanya dapat digunakan pada satu element saja, sehingga hal tersebut lah yang membuat ID jarang digunakan atau dihindari penggunannya karena tidak reusable.

4. Selektor Atribut

Selektor atribut merupakan selektor yang biasa digunakan untuk menyeleksi sebuah tag yang memiliki atribut tertentu, biasanya selektor atribut digunakan untuk menyeleksi input user untuk diberikan style. Contoh penggunannya:

5. Selektor Universal

Selektor universal merupakan selektor yang dapat menyeleksi semua tag pada html, biasanya tag ini digunakan untuk melakukan reset margin dan padding pada setiap element html sehingga kita dapat menentukan padding dan margin secara manual. Cara penggunaanya:
